# Resnet_Cifar_10

This is the practice for the implementation of ResNet. This model is trained with Cifar 10.

## Prerequisites
- Pytorch 2.0.1
- Python 3.11.4
- Window 11
- conda 23.7.4

## Training
```

# Start training with: 
python train.py -m 

# You can manually resume the training with: 
python main.py --resume --lr=0.01

```
